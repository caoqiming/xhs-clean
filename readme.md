# XHS Clean

自动检测、遮挡并辅助删除小红书低素质评论的浏览器扩展。

XHS Clean 会在小红书网页端捕获评论相关接口返回，将评论内容交给你配置的 LLM 判断是否属于低素质评论。命中的评论会被自动模糊遮挡；如果评论出现在你自己的笔记下，也可以在调试面板中手动删除，或开启自动删除。

## 主要功能

- 自动识别低素质评论：覆盖辱骂、人身攻击、阴阳怪气、性骚扰、歧视、引战、广告引流、造谣威胁等破坏讨论氛围的内容。
- 自动遮挡评论：低素质评论会在笔记评论区和通知页被模糊处理，并显示“检测到低素质评论”。按住评论区域可临时查看原文。
- 检测前预遮挡：可在 AI 检测完成前先显示“等待检测 / 检测中”，减少刷评论时直接暴露在攻击性内容下的概率。
- 当前用户笔记处理：对自己笔记下的低素质评论，可选择仅遮挡或自动删除。
- 可配置审核 Prompt：可以按自己的社区标准调整“低素质评论”的定义。
- 多 LLM 服务商：支持智谱 AI、DeepSeek、通义千问、Moonshot，以及本机 Ollama。
- 调试悬浮窗口：可查看捕获到的评论、AI 判断结果、检测失败原因，并对选中评论手动重新判断或删除。
- 本地缓存：评论审核结果会保存在浏览器本地，避免同一条评论反复请求 AI。

## 安装与运行

本项目的扩展代码在 `xhs-clean-extension` 目录中，基于 Plasmo 开发。

```bash
cd xhs-clean-extension
pnpm install
pnpm dev
```

启动开发服务后，在 Chromium 内核浏览器中加载未打包扩展：

1. 打开 `chrome://extensions` 或 `edge://extensions`。
2. 开启“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择 `xhs-clean-extension/build/chrome-mv3-dev`。

构建生产版本：

```bash
cd xhs-clean-extension
pnpm build
pnpm package
```

打包产物会生成在 `xhs-clean-extension/build` 目录下。

## 首次配置

安装扩展后，点击浏览器工具栏里的 XHS Clean 图标进入设置页。

1. 保持“启用自动检测”开启。
2. 选择 LLM 服务商。
3. 选择或填写模型名。
4. 如果服务商需要 API Key，在对应输入框中填入密钥。
5. 根据需要调整“单次 AI 请求最多评论数”和“同时进行中的 AI 请求数”。
6. 如果只想处理与自己相关的评论，开启“仅检测回复我的评论和我的笔记下的评论”。
7. 如果希望自己笔记下的低素质评论被自动删除，将“当前用户笔记低素质评论处理”设置为“删除”。

默认服务商是智谱 AI，默认模型是 `glm-4.7-flash`。

### 使用 Ollama

选择 Ollama 时，扩展会请求本机 `http://localhost:11434`。请先启动 Ollama，并确保目标模型已经拉取，例如：

```bash
ollama pull qwen2.5
ollama serve
```

如果审核时报 `403 Forbidden`，通常是 Ollama 拒绝了浏览器扩展来源。macOS 可尝试：

```bash
launchctl setenv OLLAMA_ORIGINS "chrome-extension://*"
```

然后重启 Ollama。也可以临时用下面的方式启动：

```bash
OLLAMA_ORIGINS="chrome-extension://*" ollama serve
```

## 使用方法

配置完成后，打开小红书网页版并浏览笔记评论区或通知页。扩展会自动捕获评论、调用 AI 审核，并在页面上遮挡被判定为低素质的评论。

常用设置说明：

- “启用自动检测”：总开关。关闭后不会继续自动审核或遮挡评论。
- “检测完成前遮挡评论”：开启后，尚未出结果的评论会先被遮挡并标记为“等待检测”或“检测中”。
- “仅检测回复我的评论和我的笔记下的评论”：开启后会缩小检测范围，减少 AI 调用量。
- “当前用户笔记低素质评论处理”：选择“遮挡”时仅模糊显示；选择“删除”时，自己笔记下被 AI 判定为低素质的评论会自动发起删除。
- “显示调试悬浮窗口”：开启后可在页面中查看捕获记录、AI 结果、删除按钮、清除缓存按钮和手动“AI 判断选中”操作。

## 审核规则

默认 Prompt 会把以下内容视为低素质评论：

- 人身攻击、辱骂、阴阳怪气、挑衅、嘲讽、恶意贬低。
- 性骚扰、低俗色情、恶意凝视、令人不适的身体评价。
- 仇恨、歧视、地域黑、性别攻击、群体攻击。
- 引战、恶意抬杠、明显破坏讨论氛围的攻击性表达。
- 垃圾广告、引流、诈骗、刷屏、无意义重复内容。
- 对作者或其他评论者的恶意揣测、造谣、威胁、诅咒。

默认规则不会把普通负面反馈、理性反对、礼貌纠错、中性吐槽、自嘲、玩梗或正常争议直接判定为低素质。你可以在设置页编辑审核 Prompt，让判断标准更符合自己的使用场景。

## 注意事项

- AI 判断可能误判。开启自动删除前，建议先使用“遮挡”模式观察一段时间。
- 删除功能只会对当前登录用户自己的笔记评论显示或执行；非自己笔记下的评论只能遮挡。
- API Key 保存在浏览器本地存储中，请不要在共享浏览器环境中使用自己的密钥。
- 如果 AI 自动检测连续失败，扩展会自动关闭总开关并弹出错误信息，避免持续请求失败接口。
- 小红书网页接口或页面结构变化可能导致捕获、遮挡或删除失效，需要更新扩展适配。

## 项目结构

```text
xhs-clean-extension/
  background.ts             # 调用 LLM 审核评论
  content.tsx               # 注入小红书页面，捕获评论、遮挡评论、删除评论
  extension-settings.ts     # 功能设置与默认审核 Prompt
  llm-config.ts             # LLM 服务商和模型配置
  popup.tsx                 # 扩展设置面板
  prompt.md                 # 默认审核 Prompt 参考
```
